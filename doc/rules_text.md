# Google Cloud Rapid Agent Hackathon - Official Rules & Eligibility

## Contest Period
* **Start**: May 5, 2026 at 12:00 PM PT
* **End**: June 11, 2026 at 2:00 PM PT (Deadline for submissions: June 11, 2026 @ 2:00 PM PT / June 12, 2026 @ 12:00 AM GMT+3).

## Eligibility Criteria
* Must be above the age of majority in the country/jurisdiction of residence.
* Individuals or teams (up to 4 members) can enter.
* Excludes residents of Italy, Brazil, Quebec, Cuba, Iran, Syria, North Korea, Russia, Belarus, etc.
* Must use Google Cloud AI tools and chosen track partner products.

## Core Submission Requirements
1. **Google Cloud AI tools**: Must utilize Google Cloud artificial intelligence tools (Gemini models on Agent Platform/Agent Builder, BigQuery ML, etc.).
2. **Partner MCP Integration**: Must integrate at least one participating partner's Model Context Protocol (MCP) server (Arize, Elastic, Fivetran, GitLab, MongoDB, or Dynatrace).
3. **Open Source & Public Repository**: Must provide a public, open-source code repository containing all code, assets, and instructions. The repo must include a standard open-source license file at the root.
4. **Hosted Project URL**: Must provide a public URL to the hosted project (sandbox demo with preloaded/mock data is acceptable, no login required).
5. **Video Demo**: Must include a demonstration video (~3 minutes max) showing the agent functioning on its platform.
6. **No Competitor AI dependencies**: Repository and development environment must be free of competitor AI dependencies (unapproved third-party AI assistants are not permitted in the final project code or dependencies).
