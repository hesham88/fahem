SECTION 1.3: THE MATRIX INVERSE AND DETERMINANTS

In this section, we study the criteria for a matrix to have a multiplicative inverse. 
A square matrix A of size n x n is said to be invertible (or non-singular) if there exists a matrix B of the same size such that:
A * B = B * A = I_n
where I_n is the identity matrix of order n. The matrix B is unique and is denoted by A^(-1).

THEOREM 1.3.1 (INVERSE EXISTENCE CRITERION):
A square matrix A possesses a multiplicative inverse if and only if its determinant is non-zero.
det(A) != 0

If det(A) = 0, the matrix is called a singular matrix, and its inverse does not exist. 

EXAMPLE 1:
Consider the matrix A = [[2, 3], [1, 2]].
1. Compute the determinant of A:
det(A) = (2 * 2) - (3 * 1) = 4 - 3 = 1.
Since det(A) = 1 != 0, the matrix possesses an inverse.
2. Find the inverse matrix A^(-1):
A^(-1) = (1 / det(A)) * [[2, -3], [-1, 2]] = [[2, -3], [-1, 2]].

القسم 1.3: معكوس المصفوفة والمحددات

في هذا القسم، ندرس شروط وجود المعكوس الضربي للمصفوفة.
يقال عن مصفوفة مربعة أ (A) من القياس n x n أنها قابلة للعكس (غير منفردة) إذا وجدت مصفوفة ب (B) من نفس القياس بحيث:
أ * ب = ب * أ = م_n (I_n)
حيث م_n هي مصفوفة الوحدة من الرتبة n. المصفوفة ب فريدة ويرمز لها بالرمز أ^(-1).

مبرهنة 1.3.1 (شرط وجود المعكوس):
تمتلك المصفوفة المربعة أ معكوساً ضربياً إذا وفقط إذا كان محددها لا يساوي الصفر.
det(A) != 0

إذا كان det(A) = 0، تسمى المصفوفة مصفوفة منفردة (singular matrix)، ولا يوجد لها معكوس ضربي.

مثال 1:
لتكن المصفوفة أ = [[2, 3], [1, 2]].
1. احسب محدد أ:
det(A) = (2 * 2) - (3 * 1) = 4 - 3 = 1.
بما أن المحدد det(A) = 1 != 0، فإن المصفوفة لها معكوس ضربي.
2. أوجد المصفوفة العكسية أ^(-1):
أ^(-1) = (1 / det(A)) * [[2, -3], [-1, 2]] = [[2, -3], [-1, 2]].
